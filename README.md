# jGIS_BOX
jGIS BOX is ArcGIS Pro toolbox that links your own Claude or OpenAI account to your live map. It sends your plain-language request plus the map context to the AI, then runs the returned arcpy code on your active map and shows results and new layers. Drive ArcGIS Pro by typing, not coding. Simple toolbox, no servers, bring-your-own-key. Free, v1.0.
